WooCommerce HTML Email template generator for MailChimp with drag'n'drop interface

== Description ==

WooCommerce HTML Email template generator for MailChimp with drag'n'drop interface.

This plugin allows you to create HTML email newsletter with your WooCommerce products for MailChimp email campaigns.


== Installation ==

1. Upload `woo-email-generator` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= How can I use the generated HTML template? =

You can import generated HTML template to your MailChimp account and send newsletter to your subscribers.
